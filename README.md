# 蝉印诊所管家前端资源库

## 重要说明

**本项目非源码地址，是前端资源打包目录。**

## 源码地址

- **SaaS运营平台前端源码**：[https://github.com/yiliaocrm/his-tenant-admin](https://github.com/yiliaocrm/his-tenant-admin)
- **安装引导前端源码**：[https://github.com/yiliaocrm/his-install-frontend](https://github.com/yiliaocrm/his-install-frontend)
- **后端服务源码**：[https://github.com/yiliaocrm/api](https://github.com/yiliaocrm/api)
- **机构端源码**：暂不提供（因为使用了第三方商用库，为了尊重版权暂无法开源，可付费购买）

## 介绍

本仓库为蝉印诊所管家系统的前端资源打包目录，用于存放编译后的前端静态资源文件。如需查看源代码或进行二次开发，请访问上述对应的源码仓库。

## 目录说明

本仓库下包含以下4个子目录：

- **admin**：SaaS运营平台前端打包资源
- **his**：机构端旧版本前端打包资源
- **install**：系统安装引导前端打包资源
- **web**：机构端新版本前端打包资源

## 使用说明

将本仓库中的所有目录部署到laravel中public/dist下面即可，或者上传到七牛云等云服务
